Features

Biancaa comes with some great features!

  • Mobile-friendly.
  • Featured posts slider.
  • 5 Premade colors.
  • 4 Layouts available.
  • Built-in paginations for your home, archive, and search pages.
  • Ability to upload your custom logo and favicon.
  • Ability to customize the background color.
  • Handled 2 menu areas.
  • Handled 2 widget areas.
  • Fully internationalized, so you can run the theme in any language.

Installation

Through Dashboard

  1. Log into your WordPress admin panel and go to Appearance → Themes
  2. Click the Add New button at the top.
  3. Click the Upload Theme button at the top.
  4. Click the Choose File button and navigate to the location of your biancaa.zip file.
  5. Click Install Now to begin the installation.
  6. Wait for the success message, then click Activate.

Installing Via FTP

  1. Unzip the biancaa.zip to find the theme folder.
  2. Using an FTP client to access your host web server. Go to /wp-content/themes/ and place the theme folder inside it.
  3. Log in to your WordPress admin panel and go to Appearance → Themes
  4. Locate the theme thumbnail and click the Activate link.

Custom Background

Biancaa has feature to change the default background using the native WordPress custom background function. To use it, please follow the steps below:

  1. Go to Appearance → Background.
  2. You can either set the image or color as background.
  3. Click the Save Changes button when you finish.

Custom Menu

Biancaa supports the custom menu function to allow you to setup your menus. You can set it through following steps.

  1. Go to Appearance → Menus.
  2. To create a new custom menu, click the create a new menu link, give the menu a name, and click Save Menu button.
  3. You can add pages, categories or custom link's to the menu.
  4. And, be sure to select Primary for the Theme Location before saving your menu.
  5. When you are done, click Save Menu

Please go to Codex Help for more detailed instruction.

Social Menu

Biancaa gives you the option to show your social profile icon in the footer of your site. You can set it through following steps.

  1. Go to Appearance → Menus.
  2. To create a new custom menu, click the create a new menu link, give the menu a name for example Social, and click Save Menu button.
  3. To add custom social links to this menu, use the Links box.
  4. Then, just make a link to the social media profiles you want to display.
  5. And, be sure to select Social for the Theme Location before saving your menu.
  6. When you are done, click Save Menu

Upload Your Custom Logo

You can easily to upload your custom logo to your site. Once you uploaded it, it will change the site title and description.

  1. Go to Appearance → Customize.
  2. Open the Logo section.
  3. Click the drop-down button below the Upload Logo word.
  4. Click the select a file link.
  5. Pick the logo and upload it.

Upload Your Custom Favicon

You can easily to upload your custom favicon to your site, favicon will appear in the address bar of your browser.

  1. Go to Appearance → Customize.
  2. Open the Favicon section.
  3. Click the drop-down button below the Upload Favicon word.
  4. Click the select a file link.
  5. Pick the favicon and upload it.

Featured Posts Slider

Featured posts slider is a posts based on user selected tag with a smooth slide effect, it only displayed on home page.

There are two ways to use it, please choose which one is describe your needs:

  1. For new website.
  2. For existing website.
New website
  1. Before you create a post, please create a new tag called featured.
  2. Every time you create a post and you want to display it as featured/slider, you only need select the featured tag to it.
  3. It will automatically display as featured posts slider.

Existing website

  1. Go to Appearance → Customize.
  2. Open the Featured Posts section.
  3. Look at the Tag slug for featured post setting, add your tag slug to the input box.

5 Premade Colors

Biancaa comes with 5 premade colors, but you can choose the color you want as well. You can set it through following steps.

  2. Open the Color Schemes section.

Child theme

Since Biancaa is a parent theme, you'll want to create a child theme if you plan on making any customizations. Don't know how to make a child theme? It's relatively simple. Just follow the below steps.

  • Create a theme folder in your /wp-content/themes directory called biancaa-child.
  • Then, create a style.css file within your theme folder.
  • At the top of your style.css file, add the below information.
/**
 * Theme Name: Biancaa Child
 * Theme URI: http://link-to-your-site.com
 * Description: Describe what your theme should be like.
 * Version: 0.1
 * Author: Your Name
 * Author URI: http://link-to-your-site.com
 * Tags: Add, Whatever, Tags, You, Want
 * Template: biancaa
 */

/* Importing the parent theme stylesheet. */
@import url( '../biancaa/style.css' );

From that point, you simply need to start adding your own styles. The parent theme styles will automatically be loaded for you.
